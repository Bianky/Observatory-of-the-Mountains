# Observatory-of-the-Mountains

This repostiory serves as a project management system deticated to the project *Observatory of the Mountains* developed by [Centre de Recerca Ecològica i Aplicacions Forestals (CREAF)](https://www.creaf.cat/en) and [Generalitat de Catalunya](https://web.gencat.cat/en/inici/index.html).

The suggested folder structure is as follows:

```

├── frontend
│   ├── css
│       └── style.css
│   └── js
│       └── map.js
│   └── index.html
├── backend
│   └── R
├── docs
│   ├── *additional documents useful to the repository*
├── data
│   ├── data_available.xlsx
│   └── geometry.geojson
│   └── socio-economy.json
│   └── metadata.txt
└── README.md

```
